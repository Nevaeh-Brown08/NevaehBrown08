What does "HTML" stand for?

A markup language.

HTML stands for HyperText Markup Language. It is used to structure content on the web using tags like <h1>, <p>, <a>, and <div>.

What is the CSS box model?

Think content, padding, border, margin.

The CSS box model describes how elements are sized and spaced: content area, padding around it, border, and margin that separates it from other elements.

How do you center a block with CSS?

Common with fixed width elements.

For a block with a fixed width, set margin-left:auto and margin-right:auto (shorthand margin:0 auto). For flexible centering use Flexbox: display:flex; justify-content:center; align-items:center.

What property makes an element a flex container?

Two words, first is 'display'.

Use display: flex (or display: inline-flex) on the parent to create a Flexbox container. Then use flex properties on the children for layout control.

What's the difference between padding and margin?

One is inside the border, one is outside.

Padding adds space inside the element's border (between content and border). Margin adds space outside the border (between this element and others).

How can you make text bold with CSS?

Use font-weight.

Use the font-weight property. Common values: font-weight:700 or font-weight:bold. Use semantic <strong> in HTML where appropriate for meaning.

What does :hover pseudo-class do?

A user interaction state.

The :hover pseudo-class applies styles when the user's pointing device hovers over an element. It's commonly used to change colors, show tooltips, or reveal content on hover.

How do you include a Google Font in your page?

Use <link> in the head or @import.

Add a <link href="https://fonts.googleapis.com/..." rel="stylesheet"> in the <head>, then use font-family in CSS. Avoid @import for performance if possible.

What CSS property controls stacking order?

It works with positioned elements.

The z-index property controls stacking order of positioned elements (position: relative/absolute/fixed/sticky). Higher z-index appears above lower ones.

Why use semantic HTML?

Accessibility & clarity.

Semantic HTML (e.g., <header>, <main>, <article>, <nav>) improves accessibility, SEO, and makes the document structure clearer to other developers and browsers.
