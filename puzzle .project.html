Project Overview

This assignment demonstrates a CSS Grid page with a Flexbox-powered puzzle assembled from image pieces. Place the downloaded puzzle images into an "images" folder and keep the filenames consistent with the src attributes below.

Required sections: identification, puzzle (flex), properties/notes, and completion date (footer).

Assembled Puzzle (Butterflies)

Flexbox & Grid Notes

  • Grid is used to structure the page: header, intro, puzzle area, properties, footer.
  • Flexbox is used inside .puzzle-frame (display:flex) with flex-wrap:wrap to build rows.
  • Each piece uses flex:0 0 <calculated size> so pieces are exact squares and fit exactly --cols per row.
  • The order property (.p1.. .p16) places pieces in the solved arrangement regardless of file name order.
  • The faulty piece exists in the HTML (required) but is hidden using display:none on .faulty.

Keep all puzzle images inside an images/ folder in the same project directory before uploading to GitHub.
